# 习惯/目标管理 Use Cases

## 背景

习惯和目标是孩子获得小红花的入口。习惯适合每天重复完成；目标适合一次性达成。家长需要能维护这些任务，孩子需要能查看当前任务并完成它们。

## UC1：录入习惯/目标

- 家长在“任务”页签中新增习惯或目标。
- 新增内容在独立编辑页面完成。
- 家长填写名称和奖励的小红花数量。
- 家长选择类型：
  - 习惯：每天可以完成一次，第二天重新可完成。
  - 目标：完成一次后进入历史。

## UC2：管理当前习惯/目标

- 任务页展示当前可完成的习惯和目标。
- 家长可以编辑名称、奖励小红花数量和类型。
- 家长可以删除不再需要的习惯或目标。
- 删除后的内容不再出现在当前任务列表中。

## UC3：完成习惯/目标

- 孩子可以在任务页完成当前习惯或目标。
- 完成后获得对应的小红花。
- 习惯当天完成后显示为今日已完成。
- 目标完成后从当前目标中退出。

## UC4：查看历史记录

- 任务页展示历史完成记录。
- 每条记录展示完成时的任务名称、小红花数量、状态和时间。
- 删除习惯或目标不会删除已经产生的历史完成记录。

## 验收口径

- 家长能新增、编辑、删除习惯和目标。
- 孩子能完成习惯和目标。
- 习惯和目标符合各自的完成规则。
- 支持多用户并发访问。
- 要求页面信息要准确。
