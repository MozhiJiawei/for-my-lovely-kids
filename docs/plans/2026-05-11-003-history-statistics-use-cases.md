# 历史统计 Use Cases

## 背景

历史统计用于集中查看孩子的小红花获得、心愿兑换和习惯打卡情况。任务页和心愿页只负责当前管理，历史记录统一进入统计页。

## UC1：查看总体统计

- 家长和孩子可以在“统计”页查看累计小红花、本月开花和今日开花。
- 统计页按月份展示日历。
- 可以切换上个月和下个月。

## UC2：查看某天明细

- 点击日历上的日期，可以查看当天的加花、兑换消耗和兑换心愿数量。
- 当天明细展示任务完成记录和心愿兑换记录。
- 没有记录的日期显示为空状态。

## UC3：按习惯查看打卡

- 可以选择某个习惯进行过滤。
- 选择习惯后，日历上标记这个习惯在哪些日期打了卡。
- 选择习惯后，当天明细只展示这个习惯的打卡记录。

## UC4：统一历史入口

- 任务管理页不再展示历史完成记录。
- 心愿管理页和心愿详情页不再展示历史兑换记录。
- 历史记录统一从统计页查看。

## 验收口径

- 统计页能展示总计、本月、今日数据。
- 日历能切换月份并选择日期。
- 按习惯过滤时，日期打卡标记准确。
- 任务和心愿历史记录统一集中在统计页。
- 支持多用户并发访问。
- 要求页面信息要准确。
